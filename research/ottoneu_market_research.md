# Ottoneu Market Research

**Date:** March 20, 2026
**Purpose:** Size the Ottoneu market and identify gaps in the Ottoneu tools ecosystem

---

## Key Findings Summary

### Ottoneu Platform
- **Business model:** Solo-run by Niv Shah (help@ottoneu.com)
- **League pricing:** $20-$100 per team depending on tier
  - Rookie League: $20/team (bragging rights only)
  - Starter League: $50/team ($200 / $100 / $50 prizes for 1st/2nd/3rd)
  - All-Star League: $100/team ($600 / $200 / $100 prizes)
- **Number of leagues:** Estimated 1,500-1,600 total leagues created (based on testing league IDs - league 1487 exists, 1500+ return errors)
- **Number of active users:** Unknown (no public statistics available)
- **Community:** Active forum at community.ottoneu.com with 1,596+ topics in "Managers Wanted" category alone
- **Sports:** Baseball (primary), Football, Basketball, Six Picks

### Ottoneu Community Engagement
- **Forum activity:** Moderate to high
  - 560 topics in "Ottoneu Fantasy Baseball" category
  - 519 topics in "Wishlist" (feature requests)
  - Dedicated "Tools" category exists for third-party tool discussions
- **Discord:** Not found (tested common URLs, no evidence of active Discord server)
- **Social media:** Scattered mentions on Twitter/X, no dominant Ottoneu-specific content creators
- **Podcasts:** No dedicated Ottoneu podcasts found
- **Patreon creators:** None found specifically for Ottoneu content

### Third-Party Ottoneu Tools
Based on GitHub search, limited ecosystem:
- **ottoneu-toolbox** (blue-shoes): Python UI tools for Ottoneu Baseball
- **Ottoneu** (andrewgcross): Python utilities for Ottoneu ecosystem analysis
- **ottoneu-war** (johnwelt): WAR-style player value comparisons with FanGraphs scoring
- **OttoneuBasketball** (Clarkbar36): Lineup optimizer for basketball
- **OttoneuSlackBot** (Clarkbar36): Slack integration
- **OttoneuStandings** (bkamisar): HTML standings display

**Key observation:** Most tools are single-developer projects, mostly Python, mostly analysis-focused. No dominant commercial tool exists. The ecosystem is DIY and spreadsheet-heavy.

## What Ottoneu Players Want (Based on Forum Activity)

### Most Discussed Topics
1. **Player evaluation and trades** - constant activity
2. **Keeper decisions** - "Keep or Cut" polls dominate
3. **Strategy** - auction values, roster construction
4. **Tools** - dedicated category exists, suggesting demand

### Inferred Pain Points
- **No unified tool ecosystem:** Players build their own spreadsheets and Python scripts
- **Limited analytics:** Ottoneu provides basic stats but no advanced analytics built-in
- **No surplus value calculator:** Players manually calculate value over replacement
- **Manual projection blending:** No easy way to aggregate Steamer + ZiPS + ATC
- **Lineup optimization:** No automated optimizer for daily lineup setting
- **Trade analysis:** No tool to evaluate multi-player trade value
- **Arbitration tracking:** Salary inflation and arbitration deadlines are manual

### What's Missing (Gaps in Market)
1. **SABR Points-specific content:** Nobody produces content for SABR Points format (9 of my leagues are SABR)
2. **Ottoneu-specific projections:** All projections are generic, not calibrated to Ottoneu scoring
3. **Automated lineup setting:** Roster flexibility + streaming + injuries = daily work
4. **Real-time surplus value:** Who's underpriced right now based on current projections?
5. **Visual trade analysis:** Charts comparing player values in proposed trades
6. **Mobile-first tools:** Ottoneu has mobile site but no native app, limited mobile tools
7. **Social integration:** No bot posting daily Ottoneu insights (like @UmpScorecards for umpires)

## Total Addressable Market (TAM) Estimates

### Conservative Estimate
- 1,500 Ottoneu leagues x 12 teams = 18,000 team slots
- 50% active player rate = 9,000 active Ottoneu players
- 30% would pay $20-50/year for tools = 2,700 potential customers
- Revenue potential: $54,000-$135,000/year

### Moderate Estimate
- 1,500 Ottoneu leagues x 12 teams = 18,000 team slots
- 60% active player rate = 10,800 active Ottoneu players
- 40% would pay $20-50/year for tools = 4,320 potential customers
- Revenue potential: $86,400-$216,000/year

### Optimistic Estimate
- 1,500 Ottoneu leagues x 12 teams = 18,000 team slots
- 70% active player rate = 12,600 active Ottoneu players
- 50% would pay $20-50/year for tools = 6,300 potential customers
- Revenue potential: $126,000-$315,000/year

### Assumptions & Risks
- **League number accuracy:** Based on testing URLs, 1,500 is an estimate (could be higher or lower)
- **Active rate:** Some teams may be inactive or abandoned
- **Willingness to pay:** Players already pay $20-100/team/league, suggesting high willingness
- **Competition:** Low barriers to entry, anyone can build Ottoneu tools
- **Platform risk:** Ottoneu could shut down, change API access, or build competing features
- **Market size:** Ottoneu is niche within fantasy baseball (already niche within baseball)
